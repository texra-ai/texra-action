# Security Policy

## Reporting a vulnerability

Please report security issues privately via GitHub Security Advisories on this
repository, or by email to the TeXRA maintainers. Do not open a public issue for
undisclosed vulnerabilities.

## Security model

See [docs/security.md](./docs/security.md) for how the action handles untrusted
pull-request content, read-only execution, the `require-write-access` gate, and
token scoping.
